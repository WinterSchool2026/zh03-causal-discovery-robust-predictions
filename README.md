# Causal discovery for robust predictions

## Description

In this session, participants explore how causal discovery can support robust prediction under distribution shifts. Using the example of predicting air pollution from weather-related variables across different seasons, we examine why models that perform well in one setting may fail in another. The tutorial focuses on applying simple causal discovery algorithms with a critical perspective to build predictive models that generalize more reliably. Through hands-on experimentation, participants investigate why some methods fail to capture stable relationships and how causal reasoning can help identify more robust structures.
